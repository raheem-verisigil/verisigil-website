Core · Runtime Guard

Verify Agent Action

The primary enforcement endpoint. Call this before every agent action. Returns ALLOW, DENY, or REQUIRE_HUMAN_APPROVAL in under 50ms. Every decision chained to Merkle audit trail.

POST
/v1/guard/verify
x-api-key required
ParameterTypeRequiredDescription
agent_idstringrequiredAgent passport ID — format: vsa_xxxxxx
action_typestringrequiredpayment · delete_records · deploy · data_access · send_email · api_call · tool_use · database_write · file_write · web_search
action_detailsobjectoptionalAction-specific context. For payment: {amount_usd, recipient}. For delete: {record_count}.
resourcestringoptionalResource being accessed or modified
contextstringoptionalExecution context — default: "production"
Request
Response · ALLOW
Response · HUMAN
Response · DENY
